MyBlog 部落格系統

基於 Flask 開發的現代化個人部落格平台

功能特色

📝 Markdown 支援

使用 Showdown.js 實現 Markdown 即時預覽，讓文章撰寫更加便捷高效。

🔐 安全認證

採用 bcrypt 密碼加密、CSRF 防護和 Session 管理，確保系統安全性。

🖼️ 圖片管理

支援圖片上傳、預覽和管理，輕鬆為文章添加視覺元素。

📱 響應式設計

基於 Bootstrap 5 框架，完美支援桌面、平板和手機等各種設備。

技術架構

後端技術
  • Flask 3.1.2 - Web 框架
  • SQLAlchemy - ORM 資料庫操作
  • Flask-Login - 用戶認證
  • Flask-WTF - 表單處理與 CSRF 防護
  • bcrypt - 密碼加密
前端技術
  • Bootstrap 5.3.2 - UI 框架
  • jQuery 3.7.1 - JavaScript 函式庫
  • Showdown.js 2.0.0 - Markdown 渲染
  • HTML5 / CSS3 - 網頁標準
部署方案
  • MySQL 8.0 - 關聯式資料庫
  • Docker - 容器化技術
  • Docker Compose - 多容器編排
  • Docker Volume - 資料持久化

系統架構

三層架構設計
Routes Layer（路由層）
  • user_routes.py - 前台功能
  • admin_routes.py - 後台管理
  • 處理 HTTP 請求與回應
Services Layer（服務層）
  • user_service.py - 用戶邏輯
  • article_service.py - 文章邏輯
  • image_service.py - 圖片邏輯
Models Layer（模型層）
  • user.py - 用戶模型
  • article.py - 文章模型
  • 資料庫結構定義

核心功能

  • 安全的登入/登出機制
  • bcrypt 密碼加密儲存
  • Session 狀態管理
  • @login_required 裝飾器保護路由

  • 完整的 CRUD 操作（新增、讀取、更新、刪除）
  • Markdown 編輯器與即時預覽
  • 同時儲存 Markdown 原始碼和 HTML 格式
  • 文章列表分頁顯示
  • 文章建立/更新時間記錄

  • 圖片上傳功能（支援常見格式）
  • 自動產生唯一檔名（時間戳記）
  • 圖片預覽與管理介面
  • 檔案儲存於 data/images 目錄
  • 可在文章編輯器中直接引用

安全機制

✅ 密碼安全

使用 bcrypt 進行密碼加密，加入隨機 salt，即使資料庫洩漏也無法還原明文密碼。

✅ CSRF 防護

所有 POST 請求都需要 CSRF Token 驗證，防止跨站請求偽造攻擊。

✅ SQL 注入防護

使用 SQLAlchemy ORM 進行參數化查詢，避免 SQL 注入風險。

✅ Session 管理

Flask-Login 處理用戶 Session，自動管理登入狀態和權限控制。

部署方案

Docker 容器化部署

使用 Docker Compose 進行多容器編排，實現一鍵部署：

  • myblog_server - Flask 應用容器（Port 80:8080）
  • mysql_server - MySQL 8.0 資料庫容器
  • mysql_data - 資料庫資料持久化 Volume
  • myblog_data - 圖片檔案持久化 Volume
# 一鍵啟動
docker-compose up -d

# 查看運行狀態
docker-compose ps

# 停止服務
docker-compose down

快速開始

本地開發環境設置
  1. 克隆專案 
    git clone <repository-url>
cd myblog
  2. 建立虛擬環境 
    python -m venv myenv
myenv\Scripts\activate  # Windows
source myenv/bin/activate  # Linux/Mac
  3. 安裝依賴套件 
    pip install -r requirements.txt
  4. 啟動應用程式 
    python main.py
  5. 訪問應用

    在瀏覽器中打開 http://localhost:8080
